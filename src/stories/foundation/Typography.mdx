# 서체(Typography)

기본 폰트는 Pretendard를 사용합니다. 텍스트의 두께는 Regular(400), Semibold(600) 2가지에서
활용합니다.<br />
디자인에서는 Text Style을 정의할 때 Font Size + Font Weight + Line Height를 묶어서 하나의 스타일(e.g.
caption/large)로 정의하고 이를 최소단위의 토큰이라고 여깁니다. 하지만, 실제 개발 과정에서 텍스트 스타일은
더 작은 원자 단위로 분리합니다. 예를 들어 Font Size와 Font Weight, Line Height은 각각 하나의 토큰이고,
필요에 따라 조합하여 스타일을 생성합니다.<br />

<span className="bold">Composite Token:</span> 이름이 지정된 여러 개의 자식 값으로 구성된 설계 토큰입니다.
복합 토큰은 항상 함께 적용되는 밀접하게 관련된 스타일 속성에 유용합니다. 예를 들어, 타이포그래피 스타일은
글꼴 이름, 글꼴 크기, 줄 높이 및 색상으로 구성될 수 있습니다.

<table id="guide" style={{ width: "100%", margin: "40px 0" }}>
  <thead>
    <tr>
      <th>Usage(Web)</th>
      <th style={{ width: "13%" }}>Usage(Mobile)</th>
      <th>Style</th>
      <th>Web Size</th>
      <th>Mobile Size</th>
      <th>Weight</th>
      <th>Line Height</th>
      <th>Letter spacing</th>
      <th style={{ width: "25%" }}>Description</th>
    </tr>
  </thead>
  <tbody>
    {Object.keys(typography).map((usage) => {
      const usageObj = typography[usage];
      return Object.keys(usageObj.style).map((styleNm) => {
        const styleObj = usageObj.style[styleNm];
        return (
          <tr key={`${usage}-${styleNm}`}>
            <td>{styleObj.usageWeb}</td>
            <td>{styleObj.usageMobile}</td>
            <td>{styleNm}</td>
            <td>{styleObj.webSize}</td>
            <td>{styleObj.mobileSize}</td>
            <td>{usageObj.weight}</td>
            <td>{usageObj.lineHeight}</td>
            <td>{usageObj.letterSpacing}</td>
            <td style={{ textAlign: "left" }}>{usageObj.description}</td>
          </tr>
        );
      });
    })}
  </tbody>
</table>
